# LLM Skills

Collection of agent skills for Codex and Claude.

## Installation

```bash
git clone https://github.com/skaragianis/skills.git

ln -s <path-to-repo> ~/.claude/skills
ln -s <path-to-repo> ~/.codex/skills
```

## Skills

- [clarify](./clarify/SKILL.md). Use when user wants to clarify their plan or design, or mentions "clarify".
- [prd](./prd/SKILL.md). Use when user wants to create a PRD or plan a new feature, or mentions "prd".
